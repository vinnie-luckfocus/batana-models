## Description

Create a script that converts cleaned 2D keypoint trajectories from `data/train_2d_cleaned/` and `data/val_2d_cleaned/` into the MMPose COCO-style training format. The script must preserve the 19-keypoint `coco-17-plus-bat` schema (bat keypoints at indices 17 and 18) and produce `annotations/train.json` and `annotations/val.json` alongside symlinks or copies of the source video frames/images.

## Acceptance Criteria

- [ ] Script `scripts/prepare_mmpose_dataset.py` exists and is executable.
- [ ] Output `annotations/train.json` contains valid COCO-format entries with `images`, `annotations`, and `categories` keys.
- [ ] Every annotation entry has exactly 19 keypoints (57 float values) and 19 visibility flags.
- [ ] Keypoint indices 17 and 18 correspond to `bat_knob` and `bat_barrel`, respectively.
- [ ] Training and validation splits match the number of usable videos reported by Phase 01 quality control.
- [ ] Running the script with `--verify` performs a schema check and returns exit code 0 when valid.
- [ ] Script logs the total image count and annotation count to `logs/dataset_prep.log`.
