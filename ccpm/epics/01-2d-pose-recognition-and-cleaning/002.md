# Task: Video Preprocessing and Quality Control

## Description
Build a fully automated preprocessing pipeline that ingests raw MP4/MOV files, normalizes them to 1920x1080 @ 30 fps, extracts the core swing segment, pads/trims to exactly 120 frames, and applies histogram equalization and color normalization. Enforce quality control by flagging `LOW_FPS`, `LOW_RES`, and `NO_SWING` videos as `UNREFINABLE`. Emit QC reports and trigger emergency augmentation if usable training videos fall below 50.

## Acceptance Criteria
- [ ] Preprocessing script reads from `data/raw_videos/train/` and `data/raw_videos/val/`
- [ ] Each video is resampled to 30 fps and resized to 1920x1080
- [ ] Swing completeness detector identifies load -> contact -> follow-through using wrist velocity peaks (> 30 px/frame change) combined with bat orientation change (> 15°); videos missing any phase are marked `NO_SWING` and `UNREFINABLE`
- [ ] Videos with original fps < 24 are marked `LOW_FPS` and `UNREFINABLE`
- [ ] Videos with original resolution < 720p are marked `LOW_RES` and `UNREFINABLE`
- [ ] Any video marked `UNREFINABLE` is excluded from downstream inference and preprocessing staging
- [ ] QC report JSON written to `logs/qc_report.json` with per-video status and reason codes
- [ ] Emergency augmentation flag written to `logs/emergency_augmentation.flag` if usable training samples < 50
- [ ] Preprocessed videos stored in a temporary staging area for downstream inference
