# Task: 2D Pose Inference Pipeline

## Description
Implement the RTMPose-m inference pipeline for all usable preprocessed videos. Configure the model for 19 keypoints (COCO-17 + bat_knob + bat_barrel). Output per-frame 2D coordinates and confidence scores as structured JSON files to `data/train_2d_keypoints/` and `data/val_2d_keypoints/`. Check Task 001 latency flag: if RTMPose-m > 35 ms, evaluate YOLOv8-Pose fallback; abort only if neither model achieves < 50 ms on RTX 4090.

## Acceptance Criteria
- [ ] Inference script loads RTMPose-m and processes all usable videos batch-wise
- [ ] Output JSON schema matches the `coco-17-plus-bat` format with `video_name`, `fps`, `resolution`, `keypoint_schema`, and `frames` array
- [ ] Each frame contains 19 keypoints with `id`, `x`, `y`, and `confidence`
- [ ] Latency benchmark reads Task 001 flag; if RTMPose-m > 35 ms, evaluates YOLOv8-Pose fallback; aborts only if neither model achieves < 50 ms
- [ ] All outputs written to `data/train_2d_keypoints/` and `data/val_2d_keypoints/`
