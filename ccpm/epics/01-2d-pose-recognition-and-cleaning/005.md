# Task: Trajectory Auto-Correction

## Description
Implement the tiered 2D trajectory auto-correction pipeline: cubic spline interpolation for single-frame gaps, Kalman RTS smoother for 2-5 frame gaps, optical flow + rigid bat model for gaps > 5 frames, and a global temporal total variation (TV-L2) pass. After correction, compute TC and iterate up to 3 times if < 0.82. Log all corrections to `logs/2d_cleaning_log.csv` and write cleaned JSON outputs.

## Acceptance Criteria
- [ ] Single-frame gaps corrected with cubic spline using neighboring frames
- [ ] 2-5 frame gaps corrected with Kalman Smoother (RTS)
- [ ] >5 frame gaps corrected with optical flow tracking + rigid bat model constraint (+/- 3% bat length)
- [ ] Global TV-L2 minimization applied to entire sequence
- [ ] Post-correction TC >= 0.82 for all retained videos; videos failing after 3 iterations marked `UNREFINABLE`
- [ ] `logs/2d_cleaning_log.csv` created with all correction events
- [ ] Cleaned outputs written to `data/train_2d_cleaned/` and `data/val_2d_cleaned/`
