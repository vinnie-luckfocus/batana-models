# Task: Trajectory Anomaly Detection

## Description
Build the 2D trajectory anomaly detection module that analyzes per-keypoint temporal consistency. Compute Euclidean displacements, flag jump frames using statistical thresholds (`mu + 3*sigma`), enforce optical-flow consistency for wrists and bat keypoints (> 20 px deviation flags anomaly), and apply a fallback acceleration threshold (`> 40 px`) when local statistics are unreliable.

## Acceptance Criteria
- [ ] Displacement statistics computed per keypoint across all frames
- [ ] Jump frames flagged where `d_t > mu + 3*sigma` using a sliding window
- [ ] Optical flow consistency check implemented for keypoints 9, 10, 17, 18 with 20 px threshold
- [ ] Fallback acceleration threshold (`||P_t - 2P_{t-1} + P_{t-2}|| > 40 px`) applied when sigma is unreliable
- [ ] Anomaly report generated per video listing frame IDs, keypoint IDs, and anomaly types
