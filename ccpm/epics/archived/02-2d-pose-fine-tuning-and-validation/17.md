## Description

Write automated tests for the dataset preparation script and the metric computation functions to ensure correctness, stability, and reproducibility of the pipeline's core logic.

## Acceptance Criteria

- [ ] Test file `tests/test_dataset_prep.py` exists and can be run with `pytest`.
- [ ] `test_dataset_prep.py` verifies that a sample COCO JSON output has the correct number of keypoints, correct bat keypoint names, and valid bounding boxes.
- [ ] Test file `tests/test_metrics.py` exists and can be run with `pytest`.
- [ ] `test_metrics.py` unit-tests TC, KS, and Missing Rate on synthetic keypoint arrays with known ground-truth values (e.g., perfectly smooth = high TC, one jump frame = specific KS).
- [ ] Test suite achieves at least 80% line coverage for `scripts/prepare_mmpose_dataset.py` and `scripts/validate_2d_pose.py` (excluding main blocks).
- [ ] CI or local `pytest` run passes all tests with zero failures.
- [ ] Tests use temporary directories and do not mutate files in `data/` or `models/`.
